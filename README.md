# Kubernetes Learning Portfolio

Short tagline: Demonstrating hands-on Kubernetes learning through focused projects and experiments.

## About
This repo documents my Kubernetes learning pathway: step-by-step projects, manifests, CI demos, and reflections.

## How this repo is organized
- `projects/01-getting-started/` — basic concepts, `kubectl`, pods, deployments
- `projects/02-deployments-and-rollouts/` — rolling updates, rollbacks, strategies
- `projects/03-networking/` — Services, Ingress, Network Policies
- `charts/` — simple Helm charts used in projects
- `site/` — GitHub Pages site for portfolio
